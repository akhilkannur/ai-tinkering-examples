```bash
cp master-skills-private/*.md ~/.agents/skills/
```

## ❓ Troubleshooting

### "Skills folder not found"

Create it manually:

```bash
mkdir -p ~/.claude/skills
# or
mkdir -p ~/.agents/skills
```

### "Permission denied" (Mac/Linux)

```bash
chmod +x install.sh
./install.sh
```

### "Not a collaborator" error

Make sure you've accepted the GitHub invitation sent to your email. Check spam folder.
