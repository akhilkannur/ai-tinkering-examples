# Clawdbot 101: The Complete Beginner's Guide

## What is Clawdbot?

**Clawdbot** is a free, open-source AI assistant that runs on YOUR computer (or a cheap server) and connects to apps you already use—WhatsApp, Telegram, Discord, Slack, Signal, and even iMessage.

Think of it as having a smart assistant that:
- Lives inside your messaging apps
- Remembers your past conversations
- Can organize files and notes for you
- Runs 24/7 without monthly fees (beyond AI model costs)
- Keeps your data private on your own machine

---

## Why Is Everyone Talking About Clawdbot?

Clawdbot went viral because it solves a real problem: **AI assistants that actually integrate into your life**.

Unlike ChatGPT or Claude (web-based), Clawdbot:
- **Works in your existing apps** (no new app to open)
- **Remembers context** across conversations
- **Can be proactive** (send you reminders, briefings)
- **Runs locally** (your data stays private)
- **Is free and open-source**

## Frequently Asked Questions (The Trending Ones)

### "Do I need a Mac Mini to run Clawdbot?"

**No!** This is a common myth. Clawdbot runs on:
- Any Mac (including older ones)
- Any Windows PC (via WSL2)
- Any Linux computer
- A cheap $5/month VPS server
- AWS Free Tier (literally $0)

People buy Mac Minis because they want a dedicated always-on device, but it's NOT required.

### "Is Clawdbot free?"

**The software is 100% free.** However, you need AI model access:
- Use your Claude Pro/Max subscription ($20–100/month)
- Use an Anthropic API key (pay-per-use)
- Use OpenAI API (pay-per-use)
- Use completely FREE local models (LM Studio)

### "Is Clawdbot made by Anthropic/Claude?"

**No.** Clawdbot is an independent, community-built project. It CAN use Claude (the AI model) but is not made by Anthropic.

### "Is my data safe?"

**Yes.** Everything runs on YOUR machine. Your conversations, files, and data never leave your control unless you explicitly connect to external services.

### "Can non-technical people use it?"

**Yes!** The onboarding wizard walks you through everything. If you can follow instructions and copy-paste commands, you can set it up.

## Installation Guide (Step-by-Step for Beginners)

### Step 1: Check if You Have Node.js

Open Terminal (Mac/Linux) or PowerShell (Windows) and type:

```
node --version
```

If you see `v22.x.x` or higher, skip to Step 2.

**If not installed:** Visit https://nodejs.org and download the LTS version.

### Step 2: Install Clawdbot

**Mac/Linux:** Open Terminal and paste:
```
curl -fsSL https://clawd.bot/install.sh | bash
```

**Windows (in WSL2):** Same command in your WSL terminal.

Wait 1-2 minutes for installation.

### Step 3: Run the Setup Wizard

Type:
```
clawdbot onboard
```

The wizard will ask you:
1. **Gateway mode:** Choose "Local" (first time)
2. **AI Model:** Pick Anthropic, OpenAI, or local models
3. **Authentication:** Paste your API key or use Claude subscription
4. **Messaging app:** WhatsApp, Telegram, Discord, etc.
5. **Background service:** Choose "Yes" to run automatically

### Step 4: Connect Your Messaging App

**For Telegram (easiest):**
1. Open Telegram, message `@BotFather`
2. Type `/newbot` and follow prompts
3. Copy the bot token
4. Paste into the wizard

**For WhatsApp:**
1. Scan the QR code shown in terminal
2. Done!

### Step 5: Send Your First Message

Message your bot: "Hello, what can you do?"

🎉 **Congratulations!** You have a working AI assistant.

## Connecting Your AI Model (Detailed)

### Option A: Claude Subscription (Recommended)

If you have Claude Pro/Max:

```
claude setup-token
```

Copy the token and paste during wizard setup.

### Option B: Anthropic API Key

1. Go to https://console.anthropic.com
2. Create an API key
3. Paste when wizard asks

### Option C: Free Local Models

Use LM Studio for completely free, private AI:
1. Download LM Studio
2. Download a model (like Llama 3)
3. Start the local server
4. Configure Clawdbot to use it

## Troubleshooting Common Issues

### "Command not found: clawdbot"

Restart your terminal after installation.

### "No credentials found"

Re-run `clawdbot onboard` and set up authentication again.

### "Rate limit error"

Your AI provider quota is exhausted. Wait for reset or upgrade plan.

### "Connection refused"

The gateway isn't running. Start it with:
```
clawdbot gateway start
```

### Still stuck?

Run these diagnostic commands:
```
clawdbot status
clawdbot doctor
clawdbot health
```
