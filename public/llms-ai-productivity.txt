# Moltbot (formerly Clawdbot) 101: The Complete Beginner's Guide

## What is Moltbot?

**Moltbot** is a free, open-source AI assistant that runs on YOUR computer (or a cheap server) and connects to apps you already use—WhatsApp, Telegram, Discord, Slack, Signal, and even iMessage.

Think of it as having a smart assistant that:
- Lives inside your messaging apps
- Remembers your past conversations
- Can organize files and notes for you
- Runs 24/7 without monthly fees (beyond AI model costs)
- Keeps your data private on your own machine

---

## Why Is Everyone Talking About Moltbot?

Moltbot went viral because it solves a real problem: **AI assistants that actually integrate into your life**.

Unlike ChatGPT or Claude (web-based), Moltbot:
- Lives in your **existing chats** (WhatsApp, iMessage, etc.)
- Has **long-term memory** (remembers what you said last week)
- Can **access your files** (if you let it)
- **Costs $0** if you run it with local models

It's the first "jarvis-like" experience that is easy enough for normal people to install.

---

## Frequently Asked Questions (The Trending Ones)

### "Do I need a Mac Mini to run Moltbot?"

**No!** This is a common myth. Moltbot runs on:
- Any Mac, Windows, or Linux computer
- Cloud servers (DigitalOcean, AWS, etc.)
- **Cloudflare Workers** (New!)
- Even an old laptop you leave open

People buy Mac Minis just to have a dedicated "always-on" server, but it's not required.

### "Is Moltbot free?"

Yes, the software is **100% free and open source**.

You only pay for the "brain" (the AI model):
- **Free:** Use local models (like Llama 3) via LM Studio.
- **Paid:** Connect it to Claude ($20/mo) or OpenAI (pay-per-use).

### "Is Moltbot made by Anthropic/Claude?"

**No.** Moltbot is an independent, community-built project. It CAN use Claude (the AI model) but is not made by Anthropic.

### "Is my data safe?"

**Yes.** Everything runs on **your** device. Your chats are not sent to some random company's server (unless you use a cloud AI model, in which case it goes to them, but Moltbot itself doesn't harvest data).

### "Can non-technical people use it?"

**Yes!** The onboarding wizard walks you through everything. If you can follow instructions and copy-paste commands, you can set it up.

## Installation Guide (Step-by-Step for Beginners)

### Step 1: Check if You Have Node.js

Open Terminal (Mac/Linux) or PowerShell (Windows) and type:

```
node --version
```

If you see `v22.x.x` or higher, skip to Step 2.

**If not installed:** Visit https://nodejs.org and download the LTS version.

### Step 2: Install Moltbot

Paste the install command into your terminal:

```bash
curl -fsSL https://molt.bot/install.sh | bash
```

### Step 3: Run the Setup Wizard

Type:
```bash
moltbot onboard
```

It will guide you through:
1. Selecting your AI model (Anthropic, OpenAI, or Local)
2. Choosing your messaging platform (WhatsApp, Telegram, etc.)
3. Connecting your API keys
4. Configure Moltbot to use it

### Step 4: Connect Your Messaging App

**For Telegram (easiest):**
1. Open Telegram, message `@BotFather`
2. Type `/newbot` and follow prompts
3. Copy the bot token
4. Paste into the wizard

**For WhatsApp:**
1. Scan the QR code shown in terminal
2. Done!

### Step 5: Send Your First Message

Message your bot: "Hello, what can you do?"

🎉 **Congratulations!** You have a working AI assistant.

4. Configure Moltbot to use it

## Troubleshooting Common Issues

### "Command not found: moltbot"

Restart your terminal after installation.

### "No credentials found"

Re-run `moltbot onboard` and set up authentication again.

### "Rate limit error"

Your AI provider quota is exhausted. Wait for reset or upgrade plan.

### "Connection refused"

The gateway isn't running. Start it with:
```
moltbot gateway start
```

### Still stuck?

Run these diagnostic commands:
```
moltbot status
moltbot doctor
moltbot health
```
