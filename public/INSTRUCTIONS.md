# How to Use Your AI Blueprints

Congratulations on your purchase/download of the 500+ AI Blueprints library! This library is designed to help you stop "chatting" and start "automating."

## 1. Using with AI Agents (Recommended)
These blueprints are optimized for **Agentic AI**—tools that live on your computer and can execute tasks like creating files, running shell commands, and searching your local codebase.

### Tools we recommend:
- **Claude Code:** (via Anthropic)
  - Install: `npm install -g @anthropic-ai/claude-code`
  - Usage: Open your terminal, type `claude`, and paste the blueprint.
- **Gemini CLI:** (via Google)
  - Usage: Type `gemini "your prompt here"` and paste the logic.
- **Cursor / Windsurf:** (AI Code Editors)
  - Paste the blueprint into the Composer (Cmd+I) and let it build the script/feature for you.

## 2. Using with Web-based LLMs (ChatGPT, Claude.ai)
If you are using the standard web interface, these blueprints act as **Advanced Prompt Templates**.

- **Best for:** Ideation, copywriting, and strategy analysis.
- **Limitation:** Web interfaces cannot create files on your computer or run local code. You will need to manually copy the output.
- **Usage:** Copy the "Blueprint" column from the CSV and paste it directly into the chat.

## 3. Best Practices for High Success
- **Context is King:** If a blueprint mentions a specific file or folder, make sure the AI has access to it (or paste the content of that file alongside the prompt).
- **Iterate:** If the AI fails on the first try, ask it to "sanitize the logic and try a different approach."
- **Sanitize Data:** When using these blueprints for organizational work, remember to redact sensitive personal information before pasting into public LLMs.
